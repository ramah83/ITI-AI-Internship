About This Tool

This tool generates sequence logos for genes by:

  • Fetching genetic variants (SNPs) from the UCSC genome database
  • Retrieving the corresponding DNA sequence from Ensembl
  • Creating a visual representation showing sequence conservation and variation

Example genes to try: BRCA1, TP53, EGFR, KRAS
